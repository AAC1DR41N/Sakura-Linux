# 🌸 Sakura-Linux
__*minimalism in full bloom*__

Sakura-Linux is an independent, source-based Linux distribution built for ricing, performance, and technical autonomy.

### Highlights
- **Discrete FS:** Every package lives in `/packages/`.
- **Introducing Bloom:** Use `bloom plant` to install and `bloom wilt` to remove apps.
- **SakuRC:** A simple, anti-systemd init system designed for speed.
- **Kernel 7.0.2:** Built for the bleeding edge.

### Philosophy
Sakura Linux isn't just a distro. It's a garden. I believe that your system should be as artistic as it is functional. By moving away from traditional structures, we give you total control over every everything of your installation.

### Getting Started
Sakura follows a manual, Gentoo-style installation process.
